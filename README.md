# 🧠 Memora – PREPE

Sistema de Asesoría Inteligente mediante Chatbot con IA para la Recomendación Personalizada de Regalos según Ocasión, Presupuesto y Perfil del Usuario.

---

## 📌 Descripción

Memora es una plataforma web diseñada para simplificar la elección de regalos personalizados mediante un chatbot inteligente basado en reglas.

El sistema permite:

- Asesoría interactiva mediante chatbot
- Recomendaciones personalizadas según ocasión y presupuesto
- Visualización de productos y experiencias
- Simulación de seguimiento (tracking)
- Panel de proveedor
- Sistema de gamificación (puntos e insignias)

Este proyecto fue desarrollado como parte del 8vo semestre académico.

---

## 🎯 Objetivo del Proyecto

Desarrollar una plataforma digital que integre un chatbot inteligente para mejorar la experiencia de compra y reducir la indecisión del usuario al momento de elegir un regalo.

## 🏗️ Arquitectura General

La plataforma está estructurada en los siguientes módulos:

- 🤖 Chatbot Asesor
- 🎁 Motor de Recomendaciones (basado en reglas)
- 📦 Catálogo de Productos
- 🧑‍💼 Panel de Proveedor
- 📊 Sistema de Seguimiento Simulado

---

## 🔄 Flujo del Sistema

1. El usuario interactúa con el chatbot.
2. El sistema recopila información (ocasión, presupuesto, perfil).
3. El motor de reglas filtra el catálogo.
4. Se presentan recomendaciones personalizadas.
5. El usuario puede personalizar el regalo.
6. Se simula confirmación y seguimiento del pedido.
